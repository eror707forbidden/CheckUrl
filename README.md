# CheckUrl
Check URL berfungsi sebagai alat untuk memeriksa setiap URL maupun parameter yang mengidentifikasikan status URL atau Parameter.
alat ini hanya untuk menguji suatu url skala besar pada file.txt yang tidak mungkin di uji satu persatu hanya untuk memeriksa setiap statusnya

```bash
# instalasi
git clone https://github.com/merdekasiberlab/xsscanner.git
cd CheckUrl
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
# menjalankan tools 
python3 checkURL.py
```
